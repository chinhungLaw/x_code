以上例子
xMan_qrcode({render: "table",width:200,height:200,correctLevel:0,text:text})



Nmae : qrcode.min.js
Des:把字符串转化为二维码,支持中文,英文和符号转化.
使用方法: xMan_qrcode(options) 返回 table 或者 canvas
options{
text 转二维码的数据
render 设置渲染方式 canvas:table
width 设置宽度 256
height 设置高度 256
correctLevel 纠错级别 0
background 背景颜色 #fff
foreground 前景颜色 #000
}
